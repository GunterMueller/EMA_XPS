KB-NOTIFY
=========

Syntax: (kb-notify <babylon-Format-Zeichenkette>
                   {<babylon-Format-Arg> ...})

KB-NOTIFY liefert einen Bestätigungsdialog, d.h.
Sie erhalten eine Ausgabe und können Ja drücken.

Beispiel: >(ask [Stadt _x] (kb-notify "~A" _x))
          NIL
          >(kb-notify "~{~%~A ~}"
             (instances-of 'Stadt :identifier T))
          T
