GET-EIS-ELEMENT-SIZE
====================

Syntax: (get-eis-element-size <Oberflächenelement>)

Diese Funktion liefert eine Liste mit Höhe und
Breite des entsprechenden Oberflächenelementes.

Beispiel: >(get-eis-element-size
              (get-window-element interface
                 "Drücktaste-1"))
