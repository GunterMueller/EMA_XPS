FUNCTION
========

Syntax: (function <Name>)  oder #'<name>

FUNCTION liefert die Instruktionen einer Funktion.
Falls <Name> ein Lambda-Ausdruck ist, so wird eine
Funktion erzeugt.

Beispiel:    >(function abs)
             #<SYSTEM-FUNCTION ABS>
             >(function (lambda (x y) (/ (+ x y) 2)))
             #<CLOSURE :LAMBDA (X Y) (/ (+ X Y) 2)>
