TASK-EDITOR

Der Task-Editor
===============

Bei den Tasks handelt es sich um eine Babylon2-
Erweiterung.
Mit dem Task-Editor können die Tasks editiert werden.
Zur Auswahl der betreffenden Wissensbasis ist das Feld
neben dem Label "Wissensbasis" anzuklicken. Um die Task
auszuwählen, ist das entsprechende Feld anzuklicken.
Soll eine Task erstellt werden, so ist der neue Name in
das Feld Auswahl des Auswahlmenüs einzutragen.

Das Bearbeiten-Menü:

- Diese Task übernehmen...
       Die geänderte Task wird an die Wissensbasis
       übergeben.
- Diese Task löschen...
       Die Task wird aus der Wissensbasis entfernt.
- Änderungen zurücknehmen...
       Es wird wieder die unbearbeitete Task geladen.
       Diese Option steht bei einer neuerstellten
       Task nicht zur Verfügung.

Wenn eine Task verändert wurde und diese Änderung
noch nicht in die Wissensbasis übernommen wurde, so 
wird dies durch ein Ausrufzeichen in der rechten oberen
Ecke angezeigt. 

Durch Anwählen eines Wortes in den Textfeldern der
Editoren mit dem Textcursor und anschließendes Drücken
des [Hilfe]-Knopfes [F1] kann Hilfe zur Syntax der
Babylon- und LISP-Ausdrücke abgerufen werden.
