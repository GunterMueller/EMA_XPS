SET-EIS-ELEMENT-SIZE
====================

Syntax: (set-eis-element-size <Oberflächenelement>
                              <Werte-Liste>)

Diese Funktion setzt die Größe des Oberflächen-
elementes. Die Werte müssen ganze Zahlen sein.

Beispiel: >(set-eis-element-size
              (get-window-element interface
                 "Drücktaste-1") '(10 20))
