REMOVE-DUPLICATES
=================

Syntax: (remove-duplicates <Sequenz>)

        ==> <Sequenz>

Diese Funktion entfernt die Duplikate einer Sequenz.
Sie fängt am Ende der Sequenz an.

Beispiel:      >(setq list '(a b c b a a))
               (A B C B A A)
               >(remove-duplicates list)
               (C B A) ; beginnt am Listenende
