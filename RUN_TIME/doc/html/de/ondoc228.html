KB-CONFIRM
==========

Syntax: (kb-confirm <Babylon-Format-Zeichenkette>
                    {<Babylon-Format-Arg> ...})

Diese Funktion erzeugt einen Bestätigungsdialog
in einem eigenen Fenster. Sie haben die Möglich-
keit zwischen Ja und Nein zu wählen.
Bei Bestätigung wird T geliefert, sonst NIL.

Beispiel: >(ask [Stadt _x]
                (kb-confirm "Wollen Sie die
                Stadt ~A besuchen?"
                (name-of _x)))
          Wollen Sie die Stadt Bonn besuchen?
