GET-GRAPHIC-LABEL-PICTURE
=========================

Syntax: (get-graphic-label-picture
           <Babylon-Graphik-Label>)

Diese Funktion liefert den Dateinamen des Bildes als 
Zeichenkette.

Beispiel: >(get-graphic-label-picture
             (get-window-element interface
                                 "GRAPHLBL1"))
