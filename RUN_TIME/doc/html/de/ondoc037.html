NUMERATOR
=========

Syntax: (numerator <rationale Zahl>)

        ==> <Zähler>

Diese Funktion extrahiert die Zähler einer rationalen
Zahl. Die rationale Zahl muß vom Typ integer oder
ratio sein. Bei einer ganzen Zahl ist der Zähler =
<rationale Zahl>.

Beispiel:      >(numerator 2/3)
               2
