SORT
====

Syntax: (sort <Sequenz> <Prädikat>)

Diese Funktion sortiert die Elemente einer Sequenz.
<Prädikat> ist eine Funktion, die erklärt wie
die Elemente sortiert werden müssen. SORT liefert
eine sortierte Sequenz. Diese ersetzt das Original.

Beispiel:      >(sort '(5 3 8 2 1 4) #'<)
               (1 2 3 4 5 8)
