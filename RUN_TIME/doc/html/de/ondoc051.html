ROUND
=====

Syntax: (round <nicht-komplexe Zahl>)

Diese Funktion rundet nicht-komplexe Zahlen zur
nächsten ganzen Zahl. Das Ergebnis ist die
entsprechende ganze Zahl.

Beispiel:      >(round 1.5)
               2 ;
               -0.5  ; der zweite Rückgabewert wird
                       nur unter Common Lisp unter-
                       stützt, aber nicht unter
                       Babylon Lisp!
