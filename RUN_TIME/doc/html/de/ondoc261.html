SET-MENU-SENSITIVITY
====================

Syntax: (set-menu-sensitivity 
           <Babylon-Menü> <T | NIL>)

Diese Funktion gibt an, ob das Menü empfindlich 
oder unempfindlich sein soll. NIL gibt an, daß 
das Menü unempfindlich ist.

Beispiel: >(set-menu-sensitivity
              (get-babylon-menu interface "MENU1")
              NIL)
