CONSTRAINT-EDITOR

Der Constraint-Editor
=====================

Mit dem Constraint-Editor können Constraints editiert 
und erstellt werden.
Zur Auswahl der betreffenden Wissensbasis ist das Feld
neben dem Label "Wissensbasis" anzuklicken. 
Das darunterliegenden Feld ermöglicht die Auswahl 
eines Constraints. Soll ein neues Constraint erstellt
werden, so ist der neue Name in das Feld Auswahl des 
Constraint-Auswahlmenüs einzutragen.  

Das Bearbeiten-Menü:

- Diesen Constraint übernehmen...    
       Der geänderte Constraint wird an die Wissensba-
       sis übergeben. 
- Diesen Constraint löschen...
       Dieser Constraint wird aus der Wissensbasis ent-
       fernt.
- Änderungen zurücknehmen... 
       Es wird wieder der unbearbeitete Constraint ge-
       laden.

Eine noch nicht in die Wissensbasis übertragene Ände-
rung eines Constraints wird durch ein Ausrufezeichen  
in der rechten oberen Ecke angezeigt. 

Durch Anwählen eines Wortes in den Textfeldern der
Editoren mit dem Textcursor und anschließendes Drücken
des [Hilfe]-Knopfes [F1] kann Hilfe zur Syntax der
Babylon- und LISP-Ausdrücke abgerufen werden.
