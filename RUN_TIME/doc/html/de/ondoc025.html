RATIONALP

RATIONALP
=========

Syntax: (rationalp <Objekt>)    ; Common Lisp Befehl
        (ratiop <Objekt>)       ; Babylon Lisp Befehl

        ==> T | NIL

Diese Funktion ist für die Erkennung von rationalen
Zahlen. Der Rückgabewert ist T, wenn das <Objekt>
eine rationale Zahl ist, sonst NIL.

Beispiele:     >(rationalp #C(1 2))
               NIL

               >(rationalp 3/2)
               T
