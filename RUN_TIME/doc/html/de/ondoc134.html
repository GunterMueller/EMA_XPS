LENGTH
======

Syntax : (length <Sequenz>)

         ==> <Zahl>

Diese Funktion liefert die Länge einer Sequenz. Wenn
die Sequenz ein Vektor mit einem Füllstandzeiger ist,
wird die aktive Länge zurückgeliefert.

Beispiel:      >(setq computer '("cpu" "ram"
                              "festplatte" "floppy"))
               ("cpu" "ram" "festplatte" "floppy")
               >(length computer)
               4
