EMAXPS-EXPLAIN-VIEW

emaxps-explain-view
===================

Syntax: (emaxps-explain-view)

Auswahl und Anzeige einer Regel aus einer
Regelmenge.

Aktiviert das interaktive grafische Frontend zum 
Erklärungsmechanismus der Babylon2 Inferenzmaschine.
Dadurch erscheint auch das Erklärungsfenster.
