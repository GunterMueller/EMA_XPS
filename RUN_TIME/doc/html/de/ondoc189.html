REMF
====

Syntax: (remf <Ort> <Indikator>)

Diese Funktion entfernt das Element <Indikator> und
den dazugehörigen Wert aus der Liste <Ort>. Sie
liefert T, wenn der Indikator gefunden wurde, sonst
NIL.

Beispiel:      >(setf pl '(a 1 b 2 c 3))
               (A 1 B 2 C 3)
               >(remf pl 'b)
               T
