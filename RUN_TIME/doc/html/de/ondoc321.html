LIST-TO-SET
===========

Syntax: (list-to-set <evaluates-to Liste>
                     [:TYPE <evaluates-to Typ-Name>]
                     [:TEST <Lisp-Typ: Funktion>])

        ==> #<set-of X (...)>

Diese Funktion erzeugt eine Menge mit den 
angegebenen Elementen.

Warnung: Eine Typ-Prüfung kann einen Fehler
         verursachen!

Beispiel: >(list-to-set
             (list Paris Dortmund Bruessel 
                     Bonn Berlin))
          #<set-of T (...)>
      
          >(setf *staedte*
            (list-to-set
               (list Paris Dortmund Bruessel 
                     Bonn Berlin)
               :TYPE 'STADT))
          #<set-of STADT (...)>

          >(setf *hauptstaedte*
            (list-to-set
               (list Paris Dortmund Bruessel 
                     Bonn Berlin)
               :TYPE 'HAUPTSTADT))
          #<set-of HAUPTSTADT (...)>
