CREATE
======

Syntax: (create ...)

Diese Funktion ist Bestandteil der babylon 3 
Programmierschnittstelle.

Sie hat veschiedene Syntaxen für Rahmen, Behaviors,
Constraints, Constraint-Muster, eingebundene lisp-
Dateien, Instanzen, Regeln, Regelmengen und Tasks.

Warnung: Sie wird derzeit von EMA-XPS v2.1 nicht
         unterstützt! Der Aufruf bewirkt nur einen
         Warnhinweis im Fehlerwerkzeug, die Abarbei-
         tung wird ansonsten unverändert fortgeführt.
