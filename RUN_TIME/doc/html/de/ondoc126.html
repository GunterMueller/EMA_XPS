PUSH
====

Syntax: (push <Element> <Ort>)

Diese Funktion ergänzt eine bestehende Liste (Der
Ort muß eine Liste enthalten) mit einem Element.
Die neue Liste enthält als erstes Element das neue
Element und als Rest die ursprüngliche Liste.

Beispiel:      >(setf list '(3 2 1))
               (3 2 1)
               >(push 4 list)
               (4 3 2 1)
               >list
               (4 3 2 1)
