FIND
====

Syntax: (find <Element> <Sequenz>)

Diese Funktion sucht nach einem bestimmten Element
in einer Sequenz.
Diese Funktion wird durch babylon v3 nicht
unterstützt!

Beispiel:      >(setq list '(a b c b a a))
               (A B C B A A)
               >(find 'a list)
               A
               >(find 'd list)
               NIL
