SUBSTITUTE
==========

Syntax: (substitute <neu. Elem.> <alt. Elem.> <Sequenz>)

Diese Funktion ersetzt ein altes Element einer Sequenz
durch ein neues.

Beispiel:      >(setq list '(a b c b a a))
               (A B C B A A)
               >(substitute 'e 'b list)
               (A E C E A A)
