:START
======

Syntax: :START <Wert>

:START ist ein zusätzlicher Parameter für alle
Sequenzoperationen. Der Standardwert für :START
ist 0.

Beispiel:      >(setq list '(a b c b a a))
               (A B C B A A)
               >(remove-duplicates list :START 3)
               (A B C B A)
