GET-EIS-ELEMENT-X-POSITION
==========================

Syntax: (get-eis-element-x-position
                         <Oberflächenelement>)

Diese Funktion liefert die x-Position des
Oberflächenelementes.

Beispiel: >(get-eis-element-x-position
              (get-window-element interface
                 "Drücktaste-1"))
