READ-FORM
=========

Syntax: (read-form <Eingabe-Strom> [<eof-Wert>])   
             ; Babylon Lisp
        (read <Eingabe-Strom> [<eof-Wert>])	   
             ; Common Lisp

Diese Funktion liest den nächsten symbolischen 
Ausdruck vom Terminal, bildet das entsprechende Lisp 
Objekt, und liefert das Objekt.
