CONSTRAINTNETZ-EDITOR

Der Constraintnetz-Editor
=========================

Mit dem Constraintnetz-Editor können Constraint-
netze editiert und erstellt werden. 
Zur Auswahl der betreffenden Wissensbasis ist das Feld
neben dem Label "Wissensbasis" anzuklicken. 
Das darunterliegenden Feld ermöglicht die Auswahl 
eines Constraintnetzes. Soll ein neues Constraintnetz
erstellt werden, so ist der neue Name in das Feld 
Auswahl des Constraintnetz-Auswahlmenüs einzutragen.

Das Bearbeiten-Menü:

- Dieses Constraintnetz übernehmen...    
       Das geänderte Constraintnetz wird an die 
       Wissensbasis übergeben. 
- Dieses Constraintnetz löschen...
       Dieses Constraintnetz wird aus der Wissensbasis 
       entfernt.
- Änderungen zurücknehmen... 
       Es wird wieder das unbearbeitete Constraintnetz ge-
       laden.

Eine noch nicht in die Wissensbasis übertragene Änderung
eines Constraintnetzes wird durch ein Ausrufezeichen in
der rechten oberen Ecke angezeigt. 

Durch Anwählen eines Wortes in den Textfeldern der
Editoren mit dem Textcursor und anschließendes Drücken
des [Hilfe]-Knopfes [F1] kann Hilfe zur Syntax der
Babylon- und LISP-Ausdrücke abgerufen werden.
