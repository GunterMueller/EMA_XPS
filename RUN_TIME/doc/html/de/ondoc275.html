GET-EIS-ELEMENT-WIDTH

GET-EIS-ELEMENT-WIDHT
=====================

Syntax: (get-eis-element-width <Oberflächenelement>)

Diese Funktion liefert die Breite des Oberflächen-
elementes.

Beispiel: >(get-eis-element-width
              (get-window-element interface
                 "Drücktaste-1"))
