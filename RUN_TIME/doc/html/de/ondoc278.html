SET-EIS-ELEMENT-WIDTH
=====================

Syntax: (set-eis-element-width 
           <Oberflächenelement> <Breite>)

Diese Funktion setzt die Breite des Oberflächen-
elementes. Der Wert muß eine ganze Zahl sein.

Beispiel: >(set-eis-element-width
              (get-window-element interface
                 "Drücktaste-1" 80)
