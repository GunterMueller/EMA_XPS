[HAS-PART
=========

Syntax: (ask [has-part Instanz <bql-Var>]
                     <continuation>)

HAS-PART liefert die Werte der Relationen,
die zwischen den Bestandteilen einer Instanz
bestehen.

Beispiel: >(ask [has-part SIEMENS-NIXDORF _x]
                (kb-format "~A" _x))
          #<1GB-FESTPLATTE>#<16MB-RAM>
