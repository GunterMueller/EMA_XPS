SET-EIS-ELEMENT-X-POSITION
==========================

Syntax: (set-eis-element-x-position
           <Oberflächenelement> <x-Position>)

Diese Funktion setzt die x-Position des Ober-
flächenelementes.

Beispiel: >(set-eis-element-x-position
              (get-window-element interface
                 "Drücktaste-1") 100)
