GET-EIS-ELEMENT-HEIGHT
======================

Syntax: (get-eis-element-height <Oberflächenelement>)

Diese Funktion liefert die Höhe des Oberflächen-
elementes.

Beispiel: >(get-eis-element-height
              (get-window-element interface
                 "Drücktaste-1"))
