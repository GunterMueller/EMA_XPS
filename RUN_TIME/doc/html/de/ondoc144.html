:END
====

Syntax: :END <Wert>

:END ist ein zusätzlicher Parameter für alle
Sequenzoperationen. Der Standardwert für :END ist
die Länge der Ausgangssequenz.

Beispiel:      >(setq list '(a b c b a a))
               (A B C B A A)
               >(remove-duplicates list :END 4)
               (A C B A A)
