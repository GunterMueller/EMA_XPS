SUBSTITUTE-IF
=============

Syntax: (substitute-if <neu. Elem.>
                       <Prädikat> <Sequenz>)

Diese Funktion ersetzt alle Elemente einer Sequenz,
die das Prädikat erfüllen, durch neue. Sie
liefert eine neue Sequenz.

Beispiel:      >(substitute-if 'c #'numberp '(a b 1))
               (A B C)
