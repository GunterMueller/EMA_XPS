GET-BABYLON-MENU-ITEM
=====================

Syntax: (get-babylon-menu-item <Interface>
           <Menü-Name> <Menü-Eintrag-Name>)

Diese Funktion liefert die Instanz des Babylon-
Menü-Eintrags.

Beispiel: >(get-babylon-menu-item interface
              "MENÜ1" "MENÜ-EINTRAG1")
