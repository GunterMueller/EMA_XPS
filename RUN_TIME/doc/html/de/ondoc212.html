LOOP
====

Syntax: (loop {<form> ...})

Diese Funktion führt die Formen <form> zyklisch
von links nach rechts aus. LOOP liefert nie einen
Wert. Sie kann mit Hilfe von RETURN verlassen werden.

Beispiel:      >(setf i 0)
               0
               >(loop (setf i (+ i 1)) (when (= i 5)
                      (return 'Ende)))
               ENDE
               >i
               5
