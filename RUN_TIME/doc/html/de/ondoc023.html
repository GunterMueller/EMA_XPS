NUMBERP
=======

Syntax: (numberp <Objekt>) 

        ==> T | NIL

Diese Funktion ist für die Erkennung von Zahlen. Der
Rückgabewert ist T, wenn das <Objekt> eine Zahl bzw.
eine Zahl bestimmten Typs ist, sonst NIL.

Beispiele:     >(numberp #C(1 2))
               T

               >(numberp 3)
               T

               >(numberp pi)
               T

               >(numberp "a")
               NIL
