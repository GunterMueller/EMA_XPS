POSITION-IF
===========

Syntax: (position-if <Prädikat> <Sequenz>)

Diese Funktion arbeitet ähnlich wie die POSITION-
Funktion, aber sie sucht nach dem ersten Element, mit
dem richtigen Prädikat. Sie liefert die Position
des Elementes, sonst NIL.

Beispiel:  >(position-if #'numberp '(a b c d 1 2 3 4))
           4
