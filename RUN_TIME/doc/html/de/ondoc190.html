PCONS
=====

Syntax: (pcons <Indikator> <Wert> <Liste>)
               ;Babylon Lisp Befehl
        (list* <Indikator> <Wert> <Liste>)
               ;Common Lisp Befehl

Diese Funktion erweitert die Eigenschafts-<Liste>
mit <Indikator><Wert>.

Beispiel:      >(pcons 'x 0 pl)
               (X 0 A 1 B 2 C 3)
