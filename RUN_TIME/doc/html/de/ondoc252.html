SET-TEXT-LABEL
==============

Syntax: (set-text-label 
           <Babylon-Text-Label> <Aufschrift>)

Diese Funktion schreibt einen Text auf das Text-Feld.

Beispiel: >(set-text-label
              (get-window-element interface "TXTLB1")
              "neue Eingabe:")
