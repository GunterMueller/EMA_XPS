SET-TO-LIST
===========

Syntax: (set-to-list <evaluates-to <set-of A (...)>>
                     [:EXTRACT-FN
                         <Lisp-Typ: Funktion>])

        ==> Liste der extrahierten Elemente

Die Extrahierfunktion wird auf die Elemente der Menge
angewendet.

Beispiel: >(list-to-set '(2 "Haus" 3 4 "Fenster" 5))
          #<set-of T (...)>

          >(set-to-list * :EXTRACT-FN #'stringp)
          ("Haus" "Fenster")
