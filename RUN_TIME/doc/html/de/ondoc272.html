SET-EIS-ELEMENT-Y-POSITION
==========================

Syntax: (set-eis-element-y-position
           <Oberflächenelement> <y-Position>)

Diese Funktion setzt die y-Position des Ober-
flächenelementes.

Beispiel: >(set-eis-element-y-position
              (get-window-element interface
                 "Drücktaste-1") 100)
