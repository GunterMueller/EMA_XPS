MAKE-BABYLON-MENU-ITEM
======================

Syntax: (make-babylon-menu-item <Interface>
           <Menü> <Name> <Aufschrift>
           &Optional <Aktion> &Rest <Arg>)

Diese Funktion erzeugt einen Menüeintrag für ein 
bestehendes Menü. <Menü> ist der Name des schon 
existierenden Menüs. <Name> ist der Name für den 
Menüeintrag. <Aufschrift> gibt den Eintrag an, den der 
Eintrag haben soll. <Aktion> startet eine Task, nach-
dem der Benutzer den Menü-Button gedruckt hat.

Beispiel: >(defun make-kb-interface (interface)
              (add-babylon-menu-item interface
              (make-babylon-menu-item
              (get-babylon-menu interface "menü1") 
              "menü-eintrag1" "eintrag1"
              '(start 'input-demo-task interface))))
