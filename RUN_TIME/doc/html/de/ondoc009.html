TRACER

Der Tracer
==========

Mit dem Tracer können Task-, System-, Rule-, Prolog-
und Consat(Constraint)-Aktivitäten der Babylon2-Infe-
renzmaschine angezeigt werden.

Die Menge an Ausgaben kann über Schalter bestimmt
werden.

Der [Tracer] Menüunterpunkt [Löschen] löscht den
Inhalt des Trace-Fensters.

Der Name der aktuellen Wissensbasis wird in der Menü-
kopfzeile angezeigt.
