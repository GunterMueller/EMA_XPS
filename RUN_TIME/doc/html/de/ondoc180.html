MEMBER-IF
=========

Syntax: (member-if <Prädikat> <Liste>)

Diese Funktion arbeitet wie MEMBER, aber sie sucht
nach dem ersten Element mit dem richtigen Prädikat.
<Prädikat> ist eine Funktion mit einem Argument.
MEMBER-IF liefert die Restliste oder NIL.

Beispiel:      >(member-if #'numberp '(a b c 1 2 3))
               (1 2 3)
