POP
===

Syntax: (pop <Ort>)

Diese Funktion verkürzt eine bestehende Liste (Der
Ort muß eine Liste enthalten) um ein Element. Das
ursprünglich erste Element wird als Wert geliefert.

Beispiel:      >(setf list '(4 3 2 1))
               (4 3 2 1)
               >(pop list)
               4
               >list
               (3 2 1)
