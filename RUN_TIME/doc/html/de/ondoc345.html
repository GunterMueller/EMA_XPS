:DOCUMENTATION
==============

Syntax: <documentation> ::=
                <Lisp-Typ: Zeichenkette>

Warnung: Der Erklärungseintrag wird durch 
         EMA-XPS nicht unterstützt, d.h.
         :documentation und :explanation
         werden ignoriert.

Die DOCUMENTATION ist nur für eigene Kommentare
gedacht. Sie wird nicht vom System benutzt und
liefert deshalb keinen Rückgabewert.

Beispiel: (define-frame computer
                :documentation "Beschreibt die
                Optionen des Computers")
