UNTELL
======

Syntax: (untell <einfache-Prädikation>) ::=
        (untell [<Relations-Name> <Instanz> <Wert>])

UNTELL bedeutet, daß alle übergebenen Fakten
zurückgezogen werden. Die benutzte Prädikation
muß immer vollinstanziiert sein d.h., es dürfen
keine ungebundenen logischen Variablen benutzt wer-
den. Die anonyme Variable können Sie benutzen, um
eine Instanz-Wert-Relation auf unbestimmt zu setzen.
UNTELL liefert T, wenn der Wert gelöscht wurde,
andernfalls NIL.

Beispiel: >(untell [Ort Buckingham-Palast Londen])
          T

In diesem Beispiel wird der Wert Londen gelöscht!
