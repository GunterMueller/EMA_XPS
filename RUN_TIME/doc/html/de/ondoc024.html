INTEGERP
========

Syntax: (integerp <Objekt>) 

        ==> T | NIL

Diese Funktion ist für die Erkennung von ganzen
Zahlen. Der Rückgabewert ist T, wenn das <Objekt>
eine ganze Zahl ist, sonst NIL.

Beispiele:     >(integerp 3)
               T

               >(integerp pi)
               NIL
