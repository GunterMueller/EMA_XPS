OR
==

Syntax: (or {<Form> ...})

Diese Funktion überprüft die Formen der Reihe
nach, ob eine von ihnen einen Wert ungleich NIL hat.
In diesem Fall wird dieser Wert geliefert, sonst NIL.

Beispiel:      >(or nil t)
               T
               >(or nil nil)
               NIL
