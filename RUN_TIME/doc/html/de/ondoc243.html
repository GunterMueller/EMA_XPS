MAKE-BABYLON-MENU
=================

Syntax: (make-babylon-menu <Interface>
           <Name> <Aufschrift> &Rest <Arg>)

Diese Funktion definiert ein Menü auf der Oberfläche. 
<Name> ist der Name dieses Menüs. <Aufschrift> gibt 
den Eintrag an, den das Menü haben soll. Der Menü-
eintrag erscheint am oberen Bildschirmrand.

Beispiel: >(defun make-kb-interface (interface)
              (add-babylon-menu
              interface
              (make-babylon-menu interface
              "menü1" "Optionen")))
