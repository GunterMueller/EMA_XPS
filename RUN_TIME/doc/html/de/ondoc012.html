INSTANZ-EDITOR

Der Instanz-Editor
==================

Mit dem Instanz-Editor können die Instanzen editiert 
und erstellt werden.
Zur Auswahl der betreffenden Wissensbasis ist das Feld
neben dem Label "Wissensbasis" anzuklicken. Die beiden
darunterliegenden Felder ermöglichen die Auswahl eines 
Frames sowie einer Instanz. Soll eine neue Instanz er-
stellt werden, so ist der neue Name in das Feld Auswahl 
des Instanz-Auswahlmenüs einzutragen. Neue Frames 
können mit diesem Editor nicht erstellt werden. 

Das Bearbeiten-Menü:

- Diese Instanz übernehmen...    
       Die geänderte Instanz wird an die Wissensbasis 
       übergeben. 
- Diese Instanz löschen...
       Die Instanz wird aus der Wissensbasis entfernt.
- Änderungen zurücknehmen... 
       Es wird wieder die unbearbeitete Instanz geladen.

Eine noch nicht in die Wissensbasis übertragene Änderung
einer Instanz wird durch ein Ausrufezeichen in der rechten
oberen Ecke angezeigt. 

Durch Anwählen eines Wortes in den Textfeldern der
Editoren mit dem Textcursor und anschließendes Drücken
des [Hilfe]-Knopfes [F1] kann Hilfe zur Syntax der
Babylon- und LISP-Ausdrücke abgerufen werden.
