CHAR-UPCASE
===========

Syntax: (char-upcase <Zeichen>)

Diese Funktion wandelt das angegebene Zeichen in den
entsprechenden Großbuchstaben um. Ist das nicht
möglich (z.B. bei Ziffernzeichen) so wird das
Zeichen unverändert als Wert geliefert.

Beispiel:      >(char-upcase #\a)
               #\A
