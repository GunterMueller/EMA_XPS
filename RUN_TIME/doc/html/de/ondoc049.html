FLOOR
=====

Syntax: (floor <nicht-komplexe Zahl>)

Diese Funktion rundet nicht-komplexe Zahlen ab. Das
Ergebnis ist die entsprechende ganze Zahl.

Beispiel:      >(floor 1.6)
               1 ;
               0.6     ; der zweite Rückgabewert
                         wird nur unter Common Lisp
                         unterstützt, aber nicht
                         unter Babylon Lisp!
