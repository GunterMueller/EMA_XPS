LAMBDA
======

Syntax: (lambda {<Variable> ...}
                {<Beschreibung> ...}
                {<Instruktion> ...})

Diese Funktion spezifiziert die Namen für die
Parameter einer Funktion.

Beispiel:      >((lambda (x y)
                       "arithmetischer Mittelwert" 
                       (/ (+ x y) 2)) 2 3)
               5/2
