EQUALP

EQUALP
======


Syntax: (equalp <Objekt1> <Objekt2>)
                ; Common Lisp
        (is-nearly <Objekt1> <Objekt2> [<delta>])
                ; Babylon Lisp

Diese Funktion vergleicht beide Objekte. Sie liefert
T, wenn die Objekte in etwa gleich sind. Für Zahlen
können die oben stehenden Funktionen nicht
verglichen werden: equalp arbeitet wie '='.

Beispiel:      >(equalp "a" "A")
               T
               >(equalp 1 1.01)
               NIL
