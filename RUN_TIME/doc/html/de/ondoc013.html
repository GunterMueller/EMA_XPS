BEHAVIOR-EDITOR

Der Behavior-Editor
===================

Mit dem Behavior-Editor können Behaviors editiert 
und erstellt werden.
Zur Auswahl der betreffenden Wissensbasis ist das Feld
neben dem Label "Wissensbasis" anzuklicken. Die beiden
darunterliegenden Felder ermöglichen die Auswahl eines 
Frames sowie eines Behaviors. Soll ein neues Behavior
erstellt werden, so ist der neue Name in das Feld Aus-
wahl des Behavior-Auswahlmenüs einzutragen. Neue
Frames können mit diesem Editor nicht erstellt werden. 
Mit den 3 Typ-Knöpfen "before", "primary" und "after"
ist es möglich, die Ausführungsreihenfolge gleich-
namiger Behaviors festzulegen.

Das Bearbeiten-Menü:

- Dieses Behavior übernehmen...    
       Das geänderte Behavior wird an die Wissensbasis 
       übergeben. 
- Dieses Behavior löschen...
       Das Behavior wird aus der Wissensbasis entfernt.
- Änderungen zurücknehmen... 
       Es wird wieder das unbearbeitete Behavior
       geladen.

Eine noch nicht in die Wissensbasis übertragene Ände-
rung eines Behaviors wird durch ein Ausrufezeichen in
der rechten oberen Ecke angezeigt. 

Durch Anwählen eines Wortes in den Textfeldern der
Editoren mit dem Textcursor und anschließendes Drücken
des [Hilfe]-Knopfes [F1] kann Hilfe zur Syntax der
Babylon- und LISP-Ausdrücke abgerufen werden.
