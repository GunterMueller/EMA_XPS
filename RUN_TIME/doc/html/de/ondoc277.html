SET-EIS-ELEMENT-HEIGHT
======================

Syntax: (set-eis-element-height 
           <Oberflächenelement> <Höhe>)

Diese Funktion setzt die Höhe des Oberflächen-
elementes. Der Wert muß eine ganze Zahl sein.

Beispiel: >(set-eis-element-height
              (get-window-element interface
                 "Drücktaste-1" 40)
