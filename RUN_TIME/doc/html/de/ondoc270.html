GET-EIS-ELEMENT-Y-POSITION
==========================

Syntax: (get-eis-element-y-position
                           <Oberflächenelement>)

Diese Funktion liefert die y-Position des
Oberflächenelementes.

Beispiel: >(get-eis-element-y-position
              (get-window-element interface
                 "Drücktaste-1"))
