MAKE-LIST
=========

syntax: (make-list <n>
                   [:initial-element <init.elem.>])

This function produces a list with <n> elements.
Optionally an initial-element can be set, otherwise
all elements are NIL.

example:       >(make-list 2)
               (NIL NIL)
               >(make-list 2 :initial-element 1)
               (1 1)
