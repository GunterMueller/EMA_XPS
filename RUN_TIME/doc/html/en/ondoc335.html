[HAS-PART
=========

Syntax: (ask [has-part instance <bql-var>]
                     <continuation>)

HAS-PART provides the values of the relations
that exist between the parts of an instance. 

Example: >(ask [has-part SIEMENS-NIXDORF _x]
               (kb-format "~A" _x))
         #<1GB-HARDDISK>#<16MB-RAM>
