NAME-OF
=======

syntax: (name-of <evaluates-to kb-object>)

Warning: other lisp types are passed!

This function returns the name of a knowledge 
base object.

example: >(ask [city _a-city]
               (kb-format "~%~A" (name-of _a-city)))
         PARIS
         LONDON
         NEW-YORK
         BERLIN
         NIL
