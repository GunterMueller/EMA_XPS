EMAXPS-EXPLAIN-HOW-ALL

emaxps-explain-how-all
======================

Syntax: (emaxps-explain-how-all)

Selection of any fact and explanation on 
from which rules this fact has been used.

Activates the interactive graphic front end to
the Babylon2 inference engine. This call makes
the explanation window appear.
