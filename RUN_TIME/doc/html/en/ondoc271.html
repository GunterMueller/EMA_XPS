SET-EIS-ELEMENT-X-POSITION
==========================

syntax: (set-eis-element-x-position
           <window-element> <x-position>)

This function sets the x-position of the
window-element.

example: >(set-eis-element-x-position
             (get-window-element interface
                "pushbutton-1") 100)
