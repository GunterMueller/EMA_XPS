#'

#'
==

Syntax: (function <name>)  or  #'<name>

FUNCTION returns the instructions of a function. If
<name> is a lambda-expression, then a function will
be produced.

example:     >(function abs)
             #<SYSTEM-FUNCTION ABS>
             >(function (lambda (x y) (/ (+ x y) 2)))
             #<CLOSURE :LAMBDA (X Y) (/ (+ X Y) 2)>
