SET-EIS-ELEMENT-SIZE

SET-EIS-ELEMENT-SIZE
====================

syntax: (set-eis-element-size <window-element>
                              <value-list>)

This function sets the size of the window-
element. The values have to be integers.

example: >(set-eis-element-size
             (get-window-element interface
                "pushbutton-1") '(10 20))
