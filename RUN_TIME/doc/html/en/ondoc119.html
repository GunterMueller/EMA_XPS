STRING-DOWNCASE
===============

syntax: (string-downcase <string>
               [:start <start> :end <end>])

This function returns the string with lowercase 
characters. Optionally a start- and end-position
can be set.

example:       >(string-downcase "STRING"
                   :start 1 :end 5)
               "StrinG"
