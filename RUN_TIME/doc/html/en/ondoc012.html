INSTANCE-EDITOR

The Instance-Editor
===================

Instances can be edited and created via the instance
editor.
To select the respective knowledge base you have
to click on the field with the label "Knowledge Base".
The fields below enable the selection of a frame and
a related instance. To create a new instance, enter 
the new name in the field selection of the selection
menue. New frames cannot be created using this editor.

The Edit menu:
       
- Apply this Instance...
      The modified instance will be transfered into
      the knowledge base.
- Delete this Instance...
      The instance will be removed from the knowledge
      base.
- Undo Changes
      The old instance definition is reloaded from
      the knowledge base.

Instance modifications, not yet updated in the KB, 
are displayed by an exclamation mark in the top right
corner.

Help on the syntax of babylon- and lisp-expressions 
can be requested by placing the text cursor into a
word in the text fields of the editors, followed by
a keypress on the [Help]-Button [F1].
