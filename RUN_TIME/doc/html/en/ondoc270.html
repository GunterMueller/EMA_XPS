GET-EIS-ELEMENT-Y-POSITION
==========================

syntax: (get-eis-element-y-position <window-element>)

This function returns the y-position of the
window-element.

example: >(get-eis-element-y-position
             (get-window-element interface
                "pushbutton-1"))
