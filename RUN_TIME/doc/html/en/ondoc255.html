GET-INPUT-FIELD-TEXT

GET-INPUT-FIELD-TEXT
====================

syntax: (get-input-field-text <babylon-input-field>)

This function returns the text of the input-field 
in string-form.

example: >(get-input-field-text
             (get-window-element interface
             "babylon-standard-input"))
