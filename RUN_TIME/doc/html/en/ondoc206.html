PROGN
=====

syntax: (progn {<form> *})

This function evaluates the forms sequentially, in
order, from left to right. The value, which the last
form returns, will be returned by PROGN.

example:       >(progn)
               NIL
               >(progn (+ 1 2) (* 1 2) (- 2 1))
               1
