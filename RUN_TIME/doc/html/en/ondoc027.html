FLOATP

FLOATP
======

syntax: (floatp <object>) 

        ==> T | NIL

This function verifies, whether the object is a
floating point number. A function is true (T),
when its argument is a floating point number,
otherwise it is false (NIL).

example:       >(floatp #C(1 2))
               NIL
