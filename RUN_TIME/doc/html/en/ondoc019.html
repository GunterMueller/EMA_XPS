TASK-EDITOR

The Task-Editor
===============

Tasks are a Babylon2 extension to enable event
oriented sessions. They can be edited and created
via the task editor.
To select the respective knowledge base you have to
click on the field with the label "Knowledge Base".
The field below enables the selection of a task.
To create a new task, enter the new name in the
field selection of the selection menue.

The Edit menu:

- Apply this Task...
       The changed task will be transfered into
       the knowledge base. 
- Delete this Task...
       This task will be removed from the
       knowledge base.
- Undo Changes
       The old task definition is reloaded from
       the knowledge base. This option is not
       available for a newly created task.

Task modifications, not yet updated in the KB, are
displayed by an exclamation mark in the top right
corner.

Help on the syntax of babylon- and lisp-expressions 
can be requested by placing the text cursor into a
word in the text fields of the editors, followed by
a keypress on the [Help]-Button [F1].
