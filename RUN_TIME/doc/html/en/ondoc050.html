CEILING
=======

syntax: (ceiling <non-complex number>)

This function rounds non-complex numbers. The
result is the next integer.

example:       >(ceiling 1.4)         
               2 ;
               -0.6 ; the second result only exists 
                               in Common Lisp, not
                                    in Babylon Lisp!
