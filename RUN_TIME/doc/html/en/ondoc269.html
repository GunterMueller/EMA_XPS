GET-EIS-ELEMENT-X-POSITION
==========================

syntax: (get-eis-element-x-position <window-element>)

This function returns the x-position of the
window-element.

example: >(get-eis-element-x-position
             (get-window-element interface
                "pushbutton-1"))
