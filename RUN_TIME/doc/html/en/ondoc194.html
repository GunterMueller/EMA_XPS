THE-TYPE-OF
===========

syntax: (the-type-of <object>) ; Babylon Lisp command
        (type-of <object>)     ; Common Lisp command

This function returns the type of which the object is
a member.

example:       >(the-type-of '(1 2 3))
               CONS
