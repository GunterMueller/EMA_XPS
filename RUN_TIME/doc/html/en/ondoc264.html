GET-BABYLON-MENU-ITEM
=====================

syntax: (get-babylon-menu-item <interface>
           <menu-name> <menu-item-name>)

This function returns the instance of the babylon-
menu-item.

example: >(get-babylon-menu-item interface
             "MENU1" "MENU-ITEM1")
