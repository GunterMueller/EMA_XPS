SUBSTITUTE
==========

syntax: (substitute <new elem.> <old elem.> <sequence>)

This function substitutes an old element of a sequence
by a new one.

example:       >(setq list '(a b c b a a))
               (A B C B A A)
               >(substitute 'e 'b list)
               (A E C E A A)
