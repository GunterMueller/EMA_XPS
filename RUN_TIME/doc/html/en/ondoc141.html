SUBSTITUTE-IF
=============

syntax: (substitute-if <new-item>
                       <predicate> <sequence>)

This function substitutes all elements of a sequence,
which fulfill the predicate, by new ones. It returns
a new sequence.

example:       >(substitute-if 'c #'numberp '(a b 1))
               (A B C)
