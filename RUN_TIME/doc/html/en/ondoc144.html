:END
====

syntax: :END <value>

:END is an additional parameter for all sequence-
operations. The default value for :END is the length
of the sequence.

example:       >(setq list '(a b c b a a))
               (A B C B A A)
               >(remove-duplicates list :END 4)
               (A C B A A)
