ONLY-EXPLAINABLES
=================

Syntax: (only-explainables ...)

This function is part of the babylon 3 explanation
facility.

Warning: It is not supported by EMA-XPS v2.1 at the
         moment! This function call only creates a
         warning hint in the debugger. Execution
         of the KB is not affected.
