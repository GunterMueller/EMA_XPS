ADD-OUTPUT-FIELD-TEXT
=====================

syntax: (add-output-field-text 
           <babylon-output-field> <text>)

This function adds new text to the text of 
the output-field.

example: >(add-output-field-text
             (get-window-element interface
             "babylon-standard-output")
             "This is an example")
