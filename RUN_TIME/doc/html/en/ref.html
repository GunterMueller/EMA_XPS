1. Introduction

The next pages describe preferably babylon3- commands and their state of emulation with the means of Babylon2.
EMA-XPS furthermore allows the use of the com- plete Babylon2 library as well as the complete set of CommonLISP commands.

WARNING:

Those add ons are not covered by this guidance, because the knowledge pro- grammer should preferably use the emu- lated commands of the babylon3 language!
Help on the add ons is available in the literature referenced!
