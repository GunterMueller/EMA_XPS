REVAPPEND
=========

syntax: (revappend <list1> <list2>)

REVAPPEND and APPEND work similar. REVAPPEND returns
a list, which contains all elements of the lists. It
reverses the elements of <list1>.

example:       >(revappend '(1 2 3) '(4 5 6))
               (3 2 1 4 5 6)
