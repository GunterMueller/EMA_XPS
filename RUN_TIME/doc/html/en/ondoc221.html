READ-FORM
=========

syntax: (read-form <input-stream> [<eof-value>])  
             ; Babylon Lisp
        (read <input-stream> [<eof-value>])	  
             ; Common Lisp

This function reads the next symbolic expression 
from the terminal, builds the corresponding Lisp 
object, and returns the object.
