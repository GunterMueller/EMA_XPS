CREATE
======

Syntax: (create ...)

This function is part of the babylon 3 programming
interface.

It has different syntaxs for frames, behaviors,
constraints, constraint patterns, included lisp
files, instances, rules, rule sets and tasks.

Warning: It is not supported by EMA-XPS v2.1 at the
         moment! This function call only creates a
         warning hint in the debugger. Execution
         of the KB is not affected.
