ROUND

ROUND
=====

syntax: (round <non-complex number>)

This function rounds non-complex numbers. The result
is the nearest integer.

example:       >(round 1.5)
               2 ;
               -0.5  ; the second result only exists 
                            in Common Lisp, not
                                 in Babylon Lisp!
