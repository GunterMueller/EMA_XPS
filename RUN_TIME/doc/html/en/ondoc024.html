INTEGERP

INTEGERP
========

syntax: (integerp <object>) 

        ==> T | NIL

This function verifies, whether the object is an
integer. A function is true (T), when its argument
is an integer, otherwise it is false (NIL).

examples:      >(integerp 3)
               T

               >(integerp pi)
               NIL
