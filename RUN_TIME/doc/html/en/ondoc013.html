BEHAVIOR-EDITOR

The Behavior-Editor
===================
Behaviors can be edited and created via the behavior
editor.
To select the respective knowledge base you have
to click on the field with the label "Knowledge Base".
The fields below enable the selection of a frame and
a related behavior. To create a new behavior, enter
the new name in the field selection of the selection
menue. New frames cannot be created using this
editor.
Via the 3 buttons "before", "primary" and "after" it
is possible to set the execution order of behaviors
of the same names.

The Edit menu:

- Apply this Behavior...
       The changed behavior will be transfered into
       the knowledge base.
- Delete this Behavior...
       The behavior will be removed from the knowledge
       base.
- Undo Changes
       The old behavior definition is reloaded from the
       knowledge base.

Behavior modifications, not yet updated in the KB,
are displayed by an exclamation mark in the top right
corner.

Help on the syntax of babylon- and lisp-expressions 
can be requested by placing the text cursor into a
word in the text fields of the editors, followed by
a keypress on the [Help]-Button [F1].
