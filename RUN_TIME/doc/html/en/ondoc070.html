LOG
===

syntax: (log <number> <base>)

This function calculates without <base> the natural
logarithm , otherwise the logarithm of the argument
with a base different from e.
This function is not supported by babylon v3!

examples:      >(log 2)
               0.6931472

               >(log 2 10)
               0.30102998
