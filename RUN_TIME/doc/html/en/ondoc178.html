TAILP
=====

syntax: (tailp <sublist> <list>)

This function checks, whether the sublist is part
of the list. It returns T or NIL.

example:       >(setq list '(1 2 3))
               (1 2 3)
               >(tailp (rest list) list)
               T
               >(tailp '(1 2) list)
               NIL
