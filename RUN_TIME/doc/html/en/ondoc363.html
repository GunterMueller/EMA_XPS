EMAXPS-EXPLAIN-WHY

emaxps-explain-why
==================

Syntax: (emaxps-explain-why)

Shows and gives reasons for the recent state
of the proof process (The evaluation of the
prolog clauses).

Activates the interactive graphic front end to
the Babylon2 inference engine. This call makes
the explanation window appear.
