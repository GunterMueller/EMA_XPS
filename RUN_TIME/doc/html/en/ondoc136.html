REVERSE
=======

syntax: (reverse <sequence>) 

        ==> <sequence>

This function returns the sequence in reverse order.

example:       >(setq computer '("cpu" "ram"
                                 "harddisk" "floppy"))
               ("cpu" "ram" "harddisk" "floppy")
               >(reverse computer)
               ("floppy" "harddisk" "ram" "cpu")
