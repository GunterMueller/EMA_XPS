COMPLEXP
========

syntax: (complexp <object>) 

        ==> T | NIL

This function verifies, whether the object is a
complex number. A function is true (T), when its
argument is a complex number, otherwise it is
false (NIL).

examples:      >(complexp #C(1 2))
               T

               >(complexp "a")
               NIL
