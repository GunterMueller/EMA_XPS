GET-EIS-ELEMENT-SIZE

GET-EIS-ELEMENT-SIZE
====================

syntax: (get-eis-element-size <window-element>)

This function returns a list with the height and 
the width of the window-element.

example: >(get-eis-element-size
             (get-window-element interface
                "pushbutton-1"))
