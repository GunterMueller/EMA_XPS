The Session Screen
==================

The session screen contains the graphic elements of
a session. In addition to the two standard menus
"Session" and "Help" other session specific menus 
may be created by the KB.

The "Session" menu:

- Break!
     interrupts the running session.
- Rerun
     terminates the session, destroys the session
     screen and starts the session again.
- End
     terminates the session and destroys the session
     screen.
- Grid on/off
     is a utility for the programmer. The grid
     size consists of 50 screen dots and helps
     placing graphic elements on the session screen.
