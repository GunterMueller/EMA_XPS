POSITION
========

syntax: (position <element> <sequence>)

This function searches in the sequence for the
position of a certain element. It returns the
position, when it was found, otherwise NIL.

example:       >(position 2 '(1 2 3))
               1
               >(position 4 '(1 2 3))
               NIL
