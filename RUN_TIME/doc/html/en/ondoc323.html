SAME-SETS
=========

syntax: (same-sets <evaluates-to <set-of A (...)>>
                   <evaluates-to <set-of B (...)>>
                   [:TEST <lisp-type: function>])

        ==> T | NIL                                       

This function checks, whether both sets have the
same elements.

example: >(same-sets *capital-cities* *cities*)
         NIL
 
         >(same-sets *cities*
             (union-of-sets 
                 *capital-cities* *cities*))
         T
