SET-EIS-ELEMENT-Y-POSITION
==========================

syntax: (set-eis-element-y-position
           <window-element> <y-position>)

This function sets the y-position of the
window-element.

example: >(set-eis-element-y-position
             (get-window-element interface
                "pushbutton-1") 300)
