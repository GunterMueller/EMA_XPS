PCONS
=====

syntax: (pcons <indicator> <value> <plist>)
               ;Babylon Lisp command
        (list* <indicator> <value> <plist>)
               ;Common Lisp command

This function extends the <plist> with
<indicator><value>.

example:       >(pcons 'x 0 pl)
               (X 0 A 1 B 2 C 3)
