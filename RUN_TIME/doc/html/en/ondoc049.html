FLOOR

FLOOR
=====

syntax: (floor <non-complex number>)

This function rounds off non-complex numbers. The 
result is an integer without rest. 

example:       >(floor 1.6)
               1 ;
               0.6   ; the second result only exists 
                             in Common Lisp, not
                                     in Babylon Lisp!
