:START

:START
======

syntax: :START <value>

:START is an additional parameter for all sequence-
operations. The default value for :START is 0.

example:       >(setq list '(a b c b a a))
               (A B C B A A)
               >(remove-duplicates list :START 3)
               (A B C B A)
