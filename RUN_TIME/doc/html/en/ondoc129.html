LAMBDA
======

syntax: (lambda {<variable> ...}
                {<documentation-string> ...}
                {<instruction> ...})

This function specifies the names for the parameters
of a function.

example:       >((lambda (x y)
                         (/ (+ x y) 2)) 2 3)
               5/2
